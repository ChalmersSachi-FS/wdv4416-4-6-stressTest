# Stress Test Assignment

This repository contains the files and scripts for the Stress Test Assignment using Apache JMeter.

## Project Overview

- **Backend:** Node.js Express server connected to MongoDB (running in Docker)
- **Stress Testing:** Performed with Apache JMeter CLI
- **Test Plan:** Simulates 10,000 users with 2 loops, sending POST requests to the backend

## Contents

- `stress_test_plan.jmx` — JMeter test plan file
- `run_stress_test.sh` — Shell script to run the JMeter test plan via CLI and generate reports
- `output/` — Contains test result files, logs, CSV reports, and HTML report generated by JMeter
